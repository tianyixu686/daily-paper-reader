## 摘要
在没有真实世界交互反馈或人工标注监督的情况下对大语言模型（LLM）进行后训练仍然具有挑战性，尤其是在专家标注成本高昂的专门领域。最近的无标注自演化方法通过使用模型自身的输出作为监督信号，通过额外上下文构建教师模型，并通过多数投票聚合多次回滚的预测来生成伪标签来解决这一问题。然而，这些方法并非没有缺点：基于SFT和GRPO的变体遭受域外性能下降，而基于奖励的on-policy RL则增大了校准误差。在本文中，我们提出了神经元on-policy自蒸馏（Neuron-OPSD），这是一种面向数据的无标注自蒸馏框架，利用内部神经元激活来指导训练数据选择和教师上下文构建。然后模型通过从教师分布进行on-policy蒸馏来训练，在任何阶段都不需要真实标签。在专门领域的基准测试中，Neuron-OPSD提高了域内任务性能，同时保持了跨域泛化，并减轻了先前无标注基线的校准崩溃。该框架尤其适用于在线交互或外部监督成本高昂或不可行的场景，并且在概念上区别于依赖记录的有奖励标签轨迹的离线RL方法。

## Abstract
Post-training large language models (LLMs) without real-world interaction feedback or human-labeled supervision remains challenging, particularly in specialized domains where expert annotations are costly to obtain. Recent annotation-free self-evolution methods address this by using the model's own outputs as supervision signals, constructing a teacher via additional context and aggregating predictions across multiple rollouts through majority voting to produce pseudo-labels. However, these approaches are not without drawbacks: SFT- and GRPO-based variants suffer out-of-domain performance degradation, while reward-based on-policy RL inflates calibration error. In this paper, we propose Neuron On-Policy Self-Distillation (Neuron-OPSD), a data-centric framework for annotation-free self-distillation that leverages internal neuron activations to guide both training-data selection and teacher context construction. The model is then trained via on-policy distillation from the teacher distribution, requiring no ground-truth labels at any stage. Across specialized-domain benchmarks, Neuron-OPSD improves in-domain task performance while preserving cross-domain generalization and mitigating calibration collapse over prior annotation-free baselines. This framework is particularly relevant to settings where online interaction or external supervision is costly or infeasible, and is conceptually distinct from offline RL approaches that rely on logged, reward-labeled trajectories.

### 2. 方法论：核心思想、关键技术细节、算法流程
- **核心思想**：使用LLM内部神经元激活信号（Neuron Consensus和Neuron Overlap）分别用于训练样本筛选和教师上下文构建，然后通过on-policy蒸馏（OPD）训练学生模型（零样本策略）以匹配教师（EMA更新的、带上下文条件的模型）的token级分布。
- **关键技术细节**：
  - **神经元激活提取**：对于每个输入，计算生成每个token时MLP中每个神经元的贡献分数（通过早期反嵌入），保留top-K活跃神经元，形成集合N(x)。
  - **Neuron Consensus（数据选择）**：定义s(x)=|N(x)|作为活跃神经元数量。较小的s(x)表示更高的内部共识，对应更可靠的样本。选择s(x)最低的20%作为训练子集，以过滤掉可能产生幻觉的样本。
  - **Neuron Overlap（上下文构建）**：计算样本间神经元集的Jaccard距离，为每个查询检索K个最近邻的样本文档作为few-shot示例，使教师模型获得与查询推理模式匹配的上下文，从而产生更优的token级分布。
  - **On-Policy蒸馏**：学生模型（零样本）生成轨迹，教师模型（EMA更新）在相同轨迹上计算条件分布（带few-shot上下文），最小化反向KL散度。
- **算法流程**（文字说明）：
  1. 对无标签池中每个样本，用当前模型做零样本rollout，提取神经元激活集合N(x)和计数s(x)。
  2. 根据s(x)阈值（比如最低20%）选出高质量训练子集S。
  3. 预计算所有样本对之间的Jaccard距离，对于每个被选中的样本，检索K个最相似的样本作为few-shot演示。
  4. 初始化EMA教师模型与学生相同。循环：学生采样batch，生成rollout，计算学生与教师（带上下文）的token级反向KL损失，梯度更新学生，EMA更新教师。

### 6. 主要结论与发现
- Neuron Consensus（较低活跃神经元数）与模型正确率强相关，可有效筛选高质量训练样本。
- 但仅靠激活计数选择样本并不充分：高激活样本（可能含噪声）在on-policy蒸馏下仍可带来提升，说明训练价值取决于可靠性与“锐化空间”的权衡。
- Neuron Overlap（Jaccard检索）能够构建更具信息量的few-shot上下文，当领域推理多样性高时优势明显（如化学），在多样性低的领域（如物理）优势减弱。
- Neuron-OPSD相比基线，在多数源域上取得了有竞争力或更好的域内性能，同时更好地保持了跨域泛化并减少了校准误差上升。
- 教师-学生token级熵差是域内增益的关键驱动因素：当教师显著锐化token分布时，增益最大。

### 7. 优点
- 提出了一种新颖的、完全无标注的自蒸馏框架，利用内部神经元信号同时指导数据选择和上下文构建，避免了对外部标签或更强模型的依赖。
- 方法以数据为中心，计算开销主要在预处理阶段（一次性的），训练时无额外实时推理开销。
- 实验分析透彻，不仅报告性能，还深入探究增益来源（激活-正确率关系、教师-学生熵差），提供了对机制的理解。
- 在校准方面优于基于奖励的RL方法，减少了校准膨胀，这是重要实践优势。
- 保留了跨域泛化能力，相比SFT和RL基线有显著改善。

### 8. 不足与局限
- **领域差异**：在生物和化学等域上域内改进微弱，因为在这些域中教师没有提供显著的token级锐化。
- **激活信号的局限性**：神经元激活计数虽然与幻觉相关，但仅凭此作为唯一选择标准不够充分，需要更精细的平衡。
- **检索的领域依赖**：Neuron Overlap在推理模式同质的领域（如物理）中区分度降低，退化为泛化上下文，导致增益有限。
- **模型与数据覆盖有限**：仅在Qwen3-4B模型上评估；数据集限于科学和教育，未在通用或更大规模场景测试。
- **无标签理论分析**：缺乏对神经元共识与其它不确定性信号（如熵、长度、复杂度）之间关系的深入理论分析。
- **未与替代教师构建策略比较**：仅对比了随机few-shot，未比较RAG或精心设计的prompt工程。
- **可能存在的偏差**：数据选择基于零样本激活，可能偏向模型已掌握良好的区域，忽略有潜力的困难样本。
