## 摘要
大型语言模型智能体在部分可观测、长视野的环境中运行，获取监督信息仍然是一个主要瓶颈。我们通过利用现有后训练方法中忽视的一种监督来源来解决这一问题：智能体在运行中无意中实现的成功目标。具体来说，我们引入了事后监督学习（HSL），其中辅助LLM审查每个完成的轨迹，并用智能体实际实现的所有自然语言目标重新标记该轨迹。然后，HSL将轨迹与其重新标记的目标配对，并使用这些配对进行额外的微调。为了减轻重新标记数据中的次优性，我们提出了两种HSL的学习技术：无关动作掩码和样本重新加权。我们的实验表明，HSL是灵活的，并且与现有的后训练流程兼容。它同时改进了SFT和DPO，在目标空间更多样化的长视野任务上取得了更大的收益。此外，HSL是样本高效的：在ALFWorld上，它仅使用四分之一的地面真实演示就超越了在完整数据集上训练的基线。

## Abstract
Large language model agents operate in partially observable, long-horizon settings where obtaining supervision remains a major bottleneck. We address this by utilizing a source of supervision overlooked in existing post-training methods: unintended yet successful goals embedded within agent rollouts. Specifically, we introduce Hindsight Supervised Learning (HSL), where an auxiliary LLM reviews each completed trajectory and relabels it with all of the natural-language goals the agent actually achieved. HSL then pairs the trajectory with its relabeled goals and uses these pairs for additional fine-tuning. To mitigate suboptimality in the relabeled data, we propose two learning techniques for HSL, irrelevant-action masking and sample reweighting. Our experiments show that HSL is flexible and compatible with existing post-training pipelines. It improves both SFT and DPO, with larger gains on long-horizon tasks with more diverse goal spaces. Moreover, HSL is sample-efficient: on ALFWorld, it surpasses baselines trained on the full dataset while using only one quarter of the ground-truth demonstrations.

### 1. 论文的核心问题与整体含义（研究动机和背景）
大型语言模型（LLM）智能体通常在部分可观测、长视野的交互环境中工作，例如网页浏览、工具使用、代码编程、机器人控制等。这类任务的关键瓶颈在于**获取有效的监督信号成本高昂**：人工专家演示费时费力且覆盖不足；强化学习（RL）在长视野、稀疏奖励下难以自我探索出高回报轨迹。论文观察到，智能体在执行任务时，即使未能完成原始指令，也可能在交互过程中**无意中实现了其他有效的自然语言目标**（例如，虽然没把杯子放进柜子，但把杯子放进了冰箱）。现有后训练方法普遍忽略这类隐藏的成功信号。受机器人领域的`Hindsight Experience Replay`启发，本文提出**Hindsight Supervised Learning (HSL)**，通过事后回顾，将这些“稻草”般的失败轨迹重标为“金子”般的成功示范，从而高效利用智能体自身产生的数据。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
HSL 是一个灵活的框架，可以作为附加分支与 SFT、DPO 等现有后训练方法结合。核心流程如下：

- **轨迹收集**：在当前 agent πθ 与环境交互后，收集已完成轨迹 (τT, o*)。
- **重标过程**：
  - 使用辅助 LLM (Llama-3.3-70B) 回顾轨迹，在一轮推理中识别所有实际达成的自然语言目标 {I'₁, …, I'K}（即：`{I'}= M_inst(τT, o*)`）。即使原始指令失败，也可从中提取成功的子目标。
  - 进一步让辅助 LLM 为每个动作标注是否与重标目标相关（`z_t ∈ {0,1}`），用于后续掩码。
  - 将 (轨迹段, 重标目标, 动作相关性标签) 存入动态缓冲区 D'（队列，大小 100）。
- **学习过程**：
  在重标数据上应用带掩码的 SFT 损失（`L_D'`），并同时保留在原始真实演示上的 SFT 损失，用权重 λ 混合：
  ```
  L_HSL = λ * L_D' (重标数据, 带掩码) + (1-λ) * L_train (真实数据)
  ```
  - **无关动作掩码 (Irrelevant-Action Masking)**：只对 z_t=1（与重标目标相关）的动作计算损失，避免模仿子最优行为。
  - **样本重加权 (Sample Reweighting)**：对每个示范 d，权重 `w_d = (n_d/T_d)^α * n_d`，其中 n_d 是相关动作数量，T_d 是总步数，α 平衡任务难度与最优性。优先学习更优解决更困难任务的轨迹。
- **理论分析**：论文推导了 agent 与最优策略差异的上界（Theorem 1），表明 HSL 通过缩小 `KL(π*∥πθ)` 和 `KL(πH∥πθ)` 来收紧该上界，并强调了**持续在线更新重标缓冲**的重要性——让 πH 的分布随 agent 能力提升而对齐专家分布。

### 6. 论文的主要结论与发现
- **HSL 兼容并显著提升现有方法**：尤其对长视野、多样目标空间的任务（ALFWorld）提升最大（SFT+HSL 成功率达 93.6% vs SFT 82.1%；DPO+HSL 94.8% vs DPO 82.8%），对短视野任务（WebShop）提升较小。
- **样本高效**：在 ALFWorld Unseen 上，HSL 仅用 800 个真实演示即可超过使用 3200+ 个演示的完整基线（95%+ vs 80%+），甚至在更少演示时收益更大。
- **重标记质量高**：人工抽样验证 90% 正确，且重标记比直接动作预测更容易，验证了假设。
- **关键贡献组件**：对所有已达成目标重标（而非仅失败轨迹）、无关动作掩码、样本重加权，三者缺一不可。

### 7. 优点
- **新颖思路**：首次将“事后重标所有已达成目标”系统应用于 LLM 智能体后训练，挖掘了被忽视的监督源。
- **灵活集成**：可与 SFT、DPO 等主流方法结合，不是替代而是补充。
- **理论与实践结合**：有理论界上界分析指导设计（窗口在线更新、消除次优动作的掩码）。
- **样本高效**：大幅降低对昂贵人工演示的依赖，实用性较强。
- **消融充分、分析深入**：给出了重标记质量、数据分布、多种子实验等详细分析，结论可信。

### 8. 不足与局限
- **依赖强重标模型**：目前使用 Llama-3.3-70B 进行零样本重标，未来可能需要针对复杂任务微调重标器，否则重标质量可能下降。
- **二值相关性标签粗糙**：一个动作可能相关但有细节次优性（如分次购买多件物品不如一次结账），更细粒度的评价（如奖励函数）可能更好。
- **重标记分布偏差**：某些长尾目标类型（如 "examine"）在重标数据中比例偏低，导致 agent 在这些任务上表现较差（成功率仅 54.85%）。
- **计算成本增加**：训练需额外调用大型 LLM 重标，增加 GPU 时间和 API 成本。
- **仍需要少量真实演示**：完全无监督的版本尚未探索。
- **实验覆盖有限**：仅测试了 Llama-3.2-1B 一个 agent 模型，未验证在更大模型或更多元任务上的泛化性。
- **开放世界适用性**：当前假设预定义目标空间，在无预定目标的开放场景中需额外设计目标推断方法。
